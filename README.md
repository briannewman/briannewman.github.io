# briannewman.github.io

Static GitHub Pages site.

## Page Index Convention

`index.html` is a directory of links to public pages. Keep reports and other
single-purpose pages at explicit top-level URLs, then add a link card to the
homepage.

Do not make `/` redirect to a report or turn it into a full report page. If a
report used to live at `/`, preserve that content at its named report URL before
replacing the homepage with the index.

## Hindsight Review Page

`newsletter-synthesis-hindsight.html` is a public summary of local review
artifacts from `/Users/briannewman/Projects/newsletter-intel-hindsight-review`.

Publish only aggregate grades, verdict counts, and scorecard metrics. Do not
publish raw claim packets, article IDs, corpus excerpts, gated-source text, or
local-only artifact details.

When the hindsight methodology changes, update the report page and the
`index.html` link-card description together. Keep the homepage itself as the
plain public index.
