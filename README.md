# briannewman.github.io

Static GitHub Pages site.

## Page Index Convention

`index.html` is a directory of links to public pages. Keep reports and other
single-purpose pages at explicit top-level URLs, then add a link card to the
homepage.

Do not make `/` redirect to a report or turn it into a full report page. If a
report used to live at `/`, preserve that content at its named report URL before
replacing the homepage with the index.
